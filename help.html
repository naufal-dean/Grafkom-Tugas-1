WebGL Help

Operasi yang didukung:

  • Membuat model garis

    Langkah-langkah:

    1. Pilih menu "Draw Line".
    2. Drag pada canvas mulai dari titik A (jangan lepaskan drag terlebih dahulu).
    3. Garis yang akan terbentuk dapat dilihat pada canvas. Garis tersebut akan bermula dari titik A dan mengikuti posisi mouse.
    4. Lepaskan drag untuk commit proses penggambaran model garis.
    5. Ulangi langkah di atas jika ingin menggambar model garis lainnya.

  • Membuat model persegi

    Langkah-langkah:

    1. Pilih menu "Draw Square".
    2. Drag pada canvas mulai dari titik A (jangan lepaskan drag terlebih dahulu).
    3. Persegi yang akan terbentuk dapat dilihat pada canvas dan mengikuti posisi mouse. Titik A akan menjadi titik acuan dari persegi yang akan dibentuk. Panjang sisi persegi bernilai max(delta x dari koordinat titik A dan mouse, delta y dari koordinat titik A dan mouse). Kelakuan ini sama seperti pada CorelDraw jika sedang memasukkan persegi.
    4. Lepaskan drag untuk commit proses penggambaran model persegi.
    5. Ulangi langkah di atas jika ingin menggambar model persegi lainnya.

  • Membuat model poligon

    Langkah-langkah:

    1. Pilih menu "Draw Polygon".
    2. Klik pada canvas, titik-titik pada poligon yang diinginkan.

    Sebagai catatan model baru akan terlihat di canvas setelah input titik ke-3 (saat terbentuk segitiga).

    Urutan input titik akan mempengaruhi poligon yang dihasilkan (elemen titik digambar menggunakan mode gl.TRIANGLE_FAN).

  • Menyimpan (save) model ke file

    Langkah-langkah:

    1. Klik tombol "Save Model".
    2. File penyimpanan akan terdownload dengan nama default "model.txt".

    Sebagai catatan, format penyimpanan model dapat dilihat pada bagian selanjutnya di halaman ini.

  • Memuat (load) file hasil penyimpanan model

    Langkah-langkah:

    1. Klik tombol "Choose File" untuk memilih file penyimpanan.
    2. Klik tombol "Load Model" untuk memuat model dari file yang telah dipilih.
    3. Jika file valid, model akan dimuat ke canvas.

    Sebagai catatan, format penyimpanan model dapat dilihat pada bagian selanjutnya di halaman ini.

  • Menggeser titik kontrol dengan mouse

    Langkah-langkah:

    1. Pilih menu "Drag Vertex".
    2. Drag vertex dari model apapun di canvas.

  • Mengubah panjang garis

    Langkah-langkah:

    1. Pilih menu "Drag Vertex".
    2. Drag vertex dari model garis yang ingin diubah panjangnya.

  • Mengubah ukuran sisi persegi

    Langkah-langkah (metode 1):

    1. Pilih menu "Drag Vertex".
    2. Drag vertex dari model persegi yang ingin diubah panjangnya.

    Langkah-langkah (metode 2):

    1. Pilih menu "Change Square Size".
    2. Klik model persegi yang ingin diubah ukurannya.
    3. Geser slider yang ada di bawah menu "Change Square Size".

    Sebagai catatan, nilai minimum slider dibatasi pada nilai 1.

    Selain itu, pada saat slider bernilai 100 (max), panjang sisi dari persegi yang dipilih adalah 2 (pada koordinat webgl).

  • Mengubah warna poligon

    Langkah-langkah:

    1. Pilih menu "Change Color".
    2. Pilih warna pada color picker berlabel "Select Color".
    3. Pada canvas, klik pada vertex dari model apapun yang ingin diubah warnanya.

Format file penyimpanan model:

Format seluruh file:

  • [Jumlah model yang disimpan (N)]
    [Data model 1]
    [Data model 2]
    [...]
    [Data model N]

Format "Data model" yang menyusun seluruh file seperti dituliskan di "Format seluruh file":

  • [Jumlah line representasi model terkait, termasuk line ini]
    [Konstanta mode draw webgl]
    [Tipe model]
    [Jumlah vertex (X)]
    [Data vertex 1 berupa vec2 yang menyatakan x dan y]
    [Data vertex 2 berupa vec2 yang menyatakan x dan y]
    [...]
    [Data vertex X berupa vec2 yang menyatakan x dan y]
    [Data warna 1 berupa vec4 yang menyatakan r, g, b, dan a]
    [Data warna 2 berupa vec4 yang menyatakan r, g, b, dan a]
    [...]
    [Data warna X berupa vec4 yang menyatakan r, g, b, dan a]

Catatan:

  • Jumlah line representasi model harus sesuai.
  • Pasangan konstanta mode draw webgl dan tipe model yang valid: (1, line), (6, square), (6, polygon)
